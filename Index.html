Taco Cat Exercise

or Palindrome Exercise

  • This Exercise takes a user input and determines if it is a palindrome.
  • A palindrome is a word or phrase that when reversed will be the same as the original word or phrase.
  • An example of this is: taco cat or radar or madam or nurses run.
  • A common phrase used is: A man, a plan, a canal Panama
